# **LISTS**:
### There are two types of lists: 
#### 1. Unordered List (-)
#### 2. Ordered List (numbers, alphabets, etc..,)

---
<!-- Unordered List -->
### **1. Unordered List (-)**
- item- 1
- item- 2
- item- 3
- item- 4

--> **Sub Lists in Unordered list:**
#### To implement this sub lists concept we have to use indentation(2 or 4 only)
- parent-1
  - child - 1 (2 spaces implemented)
    - child - 1.1 (4 spaces implemented)
    - child - 3 (4 spaces implemented)

---
<!-- Unordered List -->
### **2. Ordered List (numbers, alphabets(*MANUALLY*))**
1. item-1
2. item-2
3. item-3
4. item-4

--> **Sub Ordered List:**

- #### To implement this sub ordered list we have to use indentation exactly of 4 spaces 
- #### To implement this sub ordered list then also we have to use +4 spaces to the sub child (if child is containing 4 spaces and if we want to add sub child to this child then we have to +4 spaces which  **8** spaces to the sub child)

1. parent-1
    1. child-1       
    2. child-1
        1. sub-child-1
        2. sub-child-1
            1. sub-sub-child-1
            2. sub-sub-child-1
                1. sub-sub-sub-child1
                2. sub-sub-sub-child2
- From the above example we can see we can create as many as possible sub ordered lists it is only applicable to this ordered list only, because for unordered list the limit is upto to the **8spaces**
